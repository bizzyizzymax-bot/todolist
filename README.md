# To-Do List Project
Here is a To Do List app I architected from the Odin Project using the Vite Module Bundler. This app allows the user create tasks and projects for each task as well as mark of and delete tasks and projects.

## Live Demo:
https://bizzyizzymax-bot.github.io/todolist/

## Features:
-Create and delete tasks and projects

-Give tasks a name, descripttion, priority type, and due date

-Check off tasks

-Auto Save so that the user tasks and projects are saved using Javascript's localStorage function.

-How to use the Vite module bundler
